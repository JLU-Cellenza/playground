# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Added
- Initial iPaaS platform setup for Cellenza MVP demo
- Logic App module with support for multiple instances
- Service Bus module with default "inbound" queue
- API Management module (Developer SKU)
- Function App module with Consumption plan
- Storage Account module for platform configurations
- Log Analytics Workspace module for centralized logging
- Application Insights module for monitoring
- Dev and Prd environment configurations
- Comprehensive documentation and runbooks
- Cost-optimized SKUs for demo purposes
- Public access configuration for all services
- Managed Identity support across all modules
- RBAC role assignments for service communication
- Diagnostic settings connected to Log Analytics

### Configuration
- Region: France Central
- Organization: Cellenza
- Project: mvp-ipaas
- Environments: dev, prd
