# API Management Module

Creates Azure API Management instance for API gateway functionality.

## Resources
- API Management (Developer SKU)
- Managed Identity (System-Assigned)
- Diagnostic Settings

## Features
- Developer SKU for cost-optimized demo
- Built-in developer portal
- Managed Identity for backend service authentication
- Gateway logs to Log Analytics
- Public access enabled for demo

## Notes
- Deployment takes 30-45 minutes
- Developer SKU is for non-production use only
- For production, use StandardV2 or Premium SKU
