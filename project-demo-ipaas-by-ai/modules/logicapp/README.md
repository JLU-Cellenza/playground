# Logic App Standard Module

Creates Azure Logic App Standard for workflow orchestration.

## Resources
- App Service Plan (WS1 - Standard)
- Logic App Standard
- Managed Identity (System-Assigned)
- Diagnostic Settings

## Features
- Workflow runtime v4
- Node.js 18 runtime
- Managed Identity for Service Bus and other Azure services
- Application Insights integration
- Diagnostic logging to Log Analytics
